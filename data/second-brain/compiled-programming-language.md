# Compiled Programming Language

- Type of [[programming language]] that users a [[compiler]] instead of an [[interpreter]] to execute code
- The [[compiler]] translates the source code to [[machine code]] before it is run
- Examples:
  - [[Rust]]
  - [[C++]]
  - [[Java]]
  - [[Go]]
