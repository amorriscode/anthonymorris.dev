# Density

- How much mass there is per volume
  - Mass divided by volume

## Densities of Common Materials

| Material        | Density ($g/cm^3$) |
| --------------- | ------------------ |
| Gold            | 19.3               |
| Lead            | 11.3               |
| Iron            | 7.9                |
| Earth (bulk)    | 5.5                |
| Rock (typical)  | 2.5                |
| Water           | 1                  |
| Wood (typical)  | 0.8                |
| Insulating foam | 0.1                |
| Silica gel      | 0.02               |
