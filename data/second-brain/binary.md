# Binary

- Base-2 (0, 1)
- Simplifies design of computers (and related technology)
  - All information is stored using 0s and 1s
- Two's Compliment
  - Have a negative number (-7)
  - Drop the sign (7)
  - Binary (0111)
  - Flip the bits (1000)
  - Add one (1001)
