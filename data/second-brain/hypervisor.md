# Hypervisor

- Creates and manages a VM
- Manages resource allocation/sharing between host and guests

## Types

- Type 1
  - Installs on computer before any OS
  - Bare-metal hypervisor
- Type 2
  - Installs in a host OS as an application
  - Hosted hypervisor
