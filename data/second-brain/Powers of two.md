# Powers of Two
- Very useful in [[computer science]]

## How to remember all powers
- Memorize `2^0` --> `2^10`
	- `2^0 = 1`
	- `2^1 = 2`
	- `2^2 = 4`
	- `2^3 = 8`
	- `2^4 = 16`
	- `2^5 = 32`
	- `2^6 = 64`
	- `2^7 = 128`
	- `2^8 = 256`
	- `2^9 = 512`
	- `2^10 = 1064 ~= 1000`
- Approximate
```bash
2^10 ~= 1000

2^16 = 2^10 * 2^6 = 1000 * 64 = 64_000

2^32 = (2^10)^3 * 2^2 = 1B * 4 = 4B
```
